# Cybersecurity Monitoring using ELK Stack

This project simulates cybersecurity monitoring using the ELK Stack.

## Technologies Used
- Python
- Elasticsearch
- Logstash
- Kibana

## System Architecture

Python Log Generator
        ↓
activity.log
        ↓
Logstash
        ↓
Elasticsearch
        ↓
Kibana Dashboard

## Features
- Simulated log generation
- Real-time log processing
- Data storage using Elasticsearch
- Visualization using Kibana dashboards

## Visualizations
- Failed Login Attempts
- Suspicious IP Activity
- Login Activity Over Time
- Top Users by Activity
